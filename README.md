# Syringe
Simple PSR-11 compliant PHP dependency injection container

# Installation
```bash
composer require shalvah/syringe
```

# Setup
- Clone the repository
```bash
git clone https://github.xom/shalvah/syringe
```
- Install dependencies
```bash
composer install
```
- Run tests
```bash
phpunit
```
